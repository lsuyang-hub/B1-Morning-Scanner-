# T0 Pulse | T+0盘中脉冲监控系统
## 产品简介
4个时间节点（10:00 / 11:00 / 13:30 / 14:30）自动推送持仓诊断和T+0操作信号。
## 四个监控节点
| 时间 | 重点 |
|------|------|
| 10:00 | 开盘第一波方向确认 |
| 11:00 | 午盘前减仓/加仓判断 |
| 13:30 | 下午开盘方向确认 |
| 14:30 | 尾盘操作（留仓/清仓决策） |
## 信号体系
| 信号 | 条件 | 操作建议 |
|------|------|---------|
| 🔴超跌 | RSI<30 或 BB<15% 或 跌幅>2% | 可小仓补/做T降成本 |
| 🔴偏弱 | 跌幅1-2% | 持有，不加 |
| ⚪震荡 | 涨跌互现 | 观望 |
| 🟡偏强 | 涨幅1-2% | 持有，等情绪延续 |
| 🟢超涨 | RSI>70 或 BB>85% 或 涨幅>3% | 可减仓，锁定利润 |
## 内置T+0操作原则
- 恐慌杀跌时：等5-10分钟反弹再出，不要开盘立刻割
- 每日目标：每只票做2个点T，月化40%+
## 配置说明
### 1. 数据源配置
使用 codebuddy.cn 行情API（需注册获取Token）
### 2. 飞书Webhook配置
在脚本中配置 FEISHU_WEBHOOK
### 3. 持仓配置
编辑 holdings_config.py 中的 HOLD_STOCKS 字典
## 盘中调仓后
盘中换仓后立即更新持仓配置，下一个推送自动使用新持仓。
