# B1 Morning Scanner | 早盘竞价评分系统
## 🎯 产品定位
**A股短线交易者的"开盘决策神器"**
每日9:25集合竞价结束后，自动对持仓股+精选票池进行综合评分，输出持仓诊断 + 精选票池Top5 + 具体操作建议，通过飞书机器人实时推送。
## 📊 评分体系（四维打分，满分65分）
| 维度 | 满分 | 评分逻辑 |
|------|:----:|------|
| **B1超卖信号** | 25 | RSI<30 + KDJ超卖 + 量能萎缩 + 股价20日低位 = 共振强信号 |
| **竞价分** | 20 | 高开≥2%→20分 / 平开→12分 / 低开≥2%→0分 |
| **技术面** | 10 | 均线多头排列+MACD金叉+布林带支撑 |
| **筹码分布** | 10 | BB<20%=机构控盘信号，加分 |
### 综合评级标准
| 评分 | 评级 | 操作建议 |
|------|------|---------|
| ≥72分 | 🟢强烈推荐 | 积极关注，可适当追涨 |
| 57-71分 | 🟡关注 | 谨慎观察，等待更好买点 |
| <57分 | ⚪回避 | 观望为主，不建议入场 |
## 🔍 核心功能
### 1. 持仓诊断
- 读取用户持仓列表（支持多账户）
- 逐一对持仓股进行四维评分
- 输出每只股票的：综合评分、RSI、KDJ、竞价涨幅、BB%、操作建议
### 2. 精选票池推荐
从以下方向筛选Top5强势股：
- 铝业链：神火股份、天山铝业、中国铝业、云铝股份
- 化肥链：云天化、华鲁恒升、泸天化
- 风电链：大金重工、天顺风能、东方电缆
- 半导体：拓荆科技、北方华创
- 创新药：美诺华、万邦德、荣昌生物、恒瑞医药

## ⚙️ 配置说明
### 1. 数据源配置
使用 codebuddy.cn 行情API（需注册获取Token，免费额度够用）
### 2. 飞书Webhook配置
在脚本中配置 `FEISHU_WEBHOOK`（飞书机器人Webhook地址）
### 3. 持仓配置
编辑 `holdings_config.py` 中的 `HOLD_STOCKS` 字典：
```python
HOLD_STOCKS = {
"8851普通账户": [
{"code": "300442", "name": "润泽科技", "shares": 4800, "cost": 89.92},
],
"6609两融账户": [
{"code": "600875", "name": "东方电气", "shares": 2900, "cost": 36.65},
]
}
